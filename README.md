# Simulador-de-Or-amento-Free-Lancer

##  Descrição
Este projeto simula a contratação de serviços de TI, permitindo que o usuário selecione serviços e visualize o valor total em tempo real.

## Funcionalidades
- Adicionar serviços ao orçamento
- Atualização dinâmica do valor total
- Lista de serviços selecionados
- Botão para limpar seleção

##  Conceitos desenvolvidos
- Variáveis (let / const)
- Manipulação do DOM
- Eventos (onClick)
- Arrow Functions
- parseFloat()

## Ferramentas  utilizadas
- HTML
- CSS
- JavaScript

##  Como executar
1. Baixe o projeto
2. Abra o index.html no navegador
